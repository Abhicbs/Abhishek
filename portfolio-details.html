Project Description

This project involved building a dynamic three-statement financial model for UPL Ltd., including scenario analysis and sensitivity checks. The model incorporates real-world data and industry benchmarks to ensure accuracy and relevance.

Key deliverables included a DCF valuation, peer comparison, and actionable insights for strategic decision-making. The project demonstrates advanced Excel modeling, financial analysis, and presentation skills.
